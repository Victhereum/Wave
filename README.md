# Example expo.io (React Native) app with ffmpeg(FFmpegKit) integration

## Run
- `yarn install`
- `yarn prebuild`
- `yarn ios`

## Requirements

- MacOSX >= 12
- XCode >= 14
- expoSDK = 48

## Problems after upgrade expo sdk?
- `rm -rf ~/Library/Developer/Xcode/DerivedData`
- `rm -rf ~/Library/Caches/CocoaPods`
- `npx expo prebuild --clean`
